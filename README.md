# Apex Gym Customer Churn Analysis

An end-to-end data analytics project using Python for data auditing, baseline metric calculations, and feature engineering.

## Interactive Dashboard
[View Interactive Dashboard on Tableau Public](https://public.tableau.com/views/ApexGymCustomerChurnAnalysis/Dashboard1?:language=en-GB&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link) 


## Python Pipeline Features
1. Data Audit: Checks rows, columns, and tracks missing cell counts.
2. Churn Baseline: Computes the aggregate churn percentage across memberships.
3. Feature Engineering: Groups member segments into targeted age buckets.

## Deployment Details

git clone https://github.com/Naskamal99/Apex-Gym-Customer-Churn-Analysis.git
